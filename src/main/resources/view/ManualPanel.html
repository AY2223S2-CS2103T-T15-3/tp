Vimification Manual Page.

Vimification is a desktop app for managing tasks, optimized for use via a Command Line Interface

Purpose of this User Guide

The purpose of this user guide is to provide you with a comprehensive resource that will guide you through the various features and functionality of the app, so that you can efficiently manage your tasks and deadlines using VimPlanner's powerful and intuitive interface.

The guide will walk you through the process of creating and managing tasks, setting due dates, prioritizing work, and tracking progress, among other topics. It will provide you with clear and concise instructions that are easy to follow, whether you are a new Vim user or an experienced Vim enthusiast.

Additionally, it will offer troubleshooting tips and solutions to common issues that you may encounter while using VimPlanner, and provide you with a comprehensive resource that will help you optimize your productivity and stay on top of your tasks and deadlines.

Overall, we hope to empower you to streamline your daily workflows and increase your productivity. So, let's get started!

Quick Start

Some example commands you can try:

  • ":a Enhance CS2103T tp user guide" to add a task with the title "Enhance CS2103T tp user guide" to Vimification.

  • ":a Finish up weekly tp tasks -d 2023-04-01" to add a task with the title "Finish up weekly tp tasks" and deadline "2023-04-01" to Vimification.

  • ":d 2" to delete the 2nd task shown in the current list.

  • ":wq" to exit the app.

Navigation

Use h,j,k,l (or the usual arrow keys) to move to the left, down, up, right respectively.

Viewing help

Key in ":help" to access the help page.

To add "Buy grocery" as a task and inidicate its priority as not urgent

:a Buy grocery -p not_urgent

To add "Essay deadline" as a task, put a deadline with it and inidicate its priority as very urgent

:a Essay deadline -d 2023-03-31 2359 -p very_urgent

To add "Prepare OP2 slides" as a task and tag it under the labels CS2101 and presentation

:a Prepare OP2 slides -l CS2101 -l presentation
or :a Prepare OP2 slides -l presentation -l CS2101

Notes:

  • Attributes like label (-l), deadline (-d) and priority (-p) are all optional.
  • You can input these optional attributes in any order.

Delete task number 4

:d 4

Exit the application

:wq!, :q!,:wq or :q
